你好，我是 [JavaGuide](https://javaguide.cn/) 的作者 Guide。

很多同学在准备后端面试时，把大量精力放在 Java、框架和中间件上，却忽略了计算机基础。结果面试官问到 TCP 为什么是三次握手、进程和线程的区别、红黑树的应用场景这些基础问题时，反而答不上来。

计算机基础是技术面试的"必考项"，无论校招还是社招，计算机网络、操作系统、数据结构与算法都是高频考察方向。这个专栏把这些基础知识系统地整理了出来，每一篇都是面试高频考点。
